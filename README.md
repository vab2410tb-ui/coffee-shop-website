#  NAB Coffee Shop

**NAB Coffee Shop** là một ứng dụng web bán máy pha cà phê và các vật dụng liên quan đến cà phê cho phép người dùng xem sản phẩm, thêm vào giỏ hàng, đặt hàng và quản lý thông tin cá nhân. Dự án được xây dựng nhằm thực hành phát triển **Fullstack Web Application**.

##  Chức năng chính

* Đăng ký / Đăng nhập người dùng
* Xác thực OTP qua email
* Xem danh sách sản phẩm
* Thêm sản phẩm vào giỏ hàng
* Đặt hàng và quản lý đơn hàng
* Xem lịch sử đơn hàng
* Quản lý thông tin cá nhân

##  Công nghệ sử dụng

### Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router

### Backend

* Node.js
* Express.js
* RESTful API

### Database

* MongoDB

### Công cụ khác

* Git / GitHub
* Insomnia (Test API)
* Cloudinary (Quản lý hình ảnh)
* NPM

##  Cài đặt dự án

Clone repository:

```bash id="6p30cl"

```

```bash id="scgq6o"

```

Chạy dự án:

```bash id="3qd6k9"

```

Dự án có thể được deploy trên các nền tảng như **Vercel** hoặc **Render**.
